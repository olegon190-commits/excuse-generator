# DESIGN — Генератор Отмазок

## Идея

Лёгкий развлекательный сервис для демонстрации связки frontend + backend + AI API. Задача была показать полный цикл: пользовательский ввод → обработка → AI-генерация → результат с оценкой.

## Два режима, одна логика оценки

**Бесплатный режим** не делает запросов к серверу для получения отмазки — все варианты зашиты прямо в JS-коде (`EXCUSES` объект). Рейтинг считается тут же в браузере эвристикой: смотрим на длину текста, наличие цифр, конкретных мест, временных маркеров, эмоциональных слов и предложений решения — каждый признак добавляет очки к базовой оценке 7.0.

Решение сделать бесплатный режим полностью клиентским (без похода на сервер) — сознательное: это мгновенный отклик и нулевая нагрузка на бэкенд для самого частого сценария использования.

**AI-режим** — единственный который реально ходит на сервер. Промпт просит Claude сразу сгенерировать отмазку и оценить её в одном запросе, возвращая структурированный JSON `{text, rating, reason}`. Это дешевле и быстрее чем два отдельных запроса (генерация + отдельная оценка).

## Обработка ошибок AI

Если `ANTHROPIC_API_KEY` не задан — сервис не падает, а отдаёт заглушку с пометкой `mode: "stub"`. Если ключ есть, но запрос к Claude упал (сеть, лимиты, парсинг JSON) — отдаётся fallback-текст с `mode: "fallback"` и коротким объяснением в поле reason. Пользователь никогда не видит голую ошибку 500.

## Мобильная адаптация

Ключевые технические решения для корректной работы на iOS:
- `font-size: 16px` у textarea — меньшие значения вызывают автозум при фокусе
- `env(safe-area-inset-bottom)` — отступ снизу под чёлку/индикатор жестов на новых iPhone
- `touch-action: manipulation` на кнопках — убирает задержку 300ms перед реакцией на тап

## Известные ограничения

- Бесплатный тариф Render засыпает после 15 минут простоя — первый визит может занять 10-20 секунд, для этого добавлен экран загрузки со спиннером
- Rate limiting для AI-режима не реализован — для pet-проекта это приемлемо, для продакшена нужно добавить
- Нет базы данных — история сгенерированных отмазок не сохраняется, каждый визит независим
