# Генератор Отмазок 🎭

Веб-сервис который генерирует убедительные отмазки для типовых жизненных ситуаций — опоздал на работу, не сделал домашку, забыл про день рождения и другие.

Живая версия: https://excuse-generator-ivge.onrender.com

## Что внутри

Два режима работы:

**🆓 Быстрые отмазки** — шесть готовых ситуаций, для каждой несколько вариантов отмазки. Рейтинг правдоподобности считается на лету по тексту: длина, наличие конкретных деталей, временных маркеров и эмоциональных слов повышают оценку.

**✨ AI-генерация** — пользователь описывает свою ситуацию своими словами, сервис генерирует персональную отмазку через Claude API вместе с оценкой правдоподобности и коротким объяснением почему именно такая оценка.

Мобильная адаптация — интерфейс одинаково хорошо работает на телефоне и десктопе, учтены особенности iOS (safe-area для iPhone с чёлкой, шрифты от 16px чтобы не было автозума).

## Технологии

- **Backend**: Python, Flask
- **Frontend**: HTML/CSS/JS, без фреймворков
- **AI**: Claude API (Anthropic) для генерации и оценки отмазок
- **Деплой**: Render (автодеплой из GitHub)

## Быстрый старт

```bash
pip install -r requirements.txt
python app.py
```

Сервис поднимется на `http://localhost:5000`.

Для работы AI-режима нужен ключ Anthropic API — задайте переменную окружения:

```bash
export ANTHROPIC_API_KEY=your_key_here
```

Без ключа AI-режим работает в demo-режиме с заглушкой.
